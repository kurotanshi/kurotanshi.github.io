這次，在匿名函式內，我們另外宣告了一個 a，值為 100，因為其作用域的關係，所以 console 的結果會是 100。

接著，問題來了，如果我們在 var a = 100; 之前去取值，會發生什麼事呢？

答案是，第一次的 console.log(a) 會印出 undefined，而第二次會出現 100。
